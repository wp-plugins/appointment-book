Uses the Book Appointment plugin to add events,view events,manage events on particular date.

== Description ==

This plugin combines the power of FullCalendar and WordPress to present your events in a calendar format.

= Features =

Backend Admin:-
1.	Add new appointment
2.	Manage appointments (Approve and reject)
3.	Show list of appointments and edit appointments.
4.	Add new appointments and edit appointments from calendar.
5.	Email send
6.  Setting options

Frontend User:-
1.	Add new appointment
2.	Show booked appointments on calendar
3.	Manage appointments 	
4.	Email send

== Installation ==

1. Upload `appointment-book` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the settings from Appointment Book Options->Settings
4. Create a page or a post and insert the shortcode [calendershow] into the text.
5. Create a page or a post and insert the shortcode [appointmentshow] into the text
6. Create a page or post and insert the shortcode [manage_front_appointmentslist] into the text.

== Frequently Asked Questions ==

= How can I add Calendar to my page? =

You should put the shortcode [calendershow] into your page or post.

= How can I add Register form to my page?

You shoul put the shortcode [user_register] into your page or post.
